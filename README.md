# 🐣 LifeGotchi

한국형 **인생 · 커리어 다마고치** 웹앱. 아기 캐릭터를 만들어 밥 · 공부 · 운동 · 자기개발을
주기적으로 직접 눌러 키우고, 대학 진학 → 취업 → 업무평가 → 연봉협상 → 승진 → 결혼 · 출산 →
은퇴 · 엔딩 → **2세대 상속**까지 한 인생을 경험합니다.

> 단순 방치형이 아닙니다. **페이지를 켜두고 정해진 주기마다 직접 액션을 눌러야** 성장해요.
> 게임 1년 = 현실 9분(조절 가능), 60세 자연사까지 약 9시간 — 하루 근무시간 한 판.

비주얼은 **2D 픽셀 다마고치** 감성입니다. 본문은 **Pretendard**, 제목·라벨·숫자는 한국어 픽셀
폰트 **Galmuri**. 캐릭터는 16×20 코드 매트릭스를 부품 레이어로 조립하는 **사람형 픽셀
스프라이트**(외부 이미지 0개, div 그리드 렌더)이고, 성장 단계·직업·시간대·계절에 따라 달라지는
**픽셀 방** 안에서 살아갑니다.

## ✨ 주요 기능

**돌봄 루프**

- 컨디션 11종(배고픔/에너지/기분/건강/집중/청결/수면질/자신감/스트레스/번아웃/체중) 실시간 감쇠
- 음식 14종(칼로리 등급·불량식품·과식 페널티), 유산소/근력 운동, 30분 공부 세션, 씻기, 칭찬하기(자신감 관리)
- **효과 가시화**: 모든 액션 버튼에 효과 미리보기(어떤 컨디션이 얼마나 변하는지), 실행 후엔
  대성공/부진 배수·과식 페널티까지 합산된 **실제 적용 수치**를 토스트로 표시
- 모든 컨디션이 실제로 작동: 청결이 낮으면 냄새 연출+효율 저하, 수면의 질·운동/식사 실천은 업무평가에 반영

**미니게임 / 행운 / 뽑기**

- 슬롯머신 🎰 · 행운 뽑기 🎁 — 쿨타임 없이 **체력을 소모해** 즉시 플레이 (6살부터)
- 슬롯 당첨금은 저축으로, 뽑기 희귀 캡슐은 스탯 포인트 지급
- **상점 뽑기**: 옷 🎽 · 방 인테리어 📦 · 자동차 🎟️ 는 직접 구매 대신 **확률 뽑기**로 획득
  — 고정 티켓값, 미소장 아이템 중 랜덤(중복 없음), 상위 가격대가 레어
- **행운 스탯**: 꽝이어도 뽑을 때마다 쌓이고(상한 있음), 당첨·레어 확률과 일반 활동의
  대성공 확률에 보너스

**인생 진행**

- 성장 단계 10종(아기→…→은퇴 준비 55세) · 연간 리뷰/시험 · 레벨업 스탯 포인트 배분
- 대학 4티어(등록금·학자금대출 연 3% 복리·자동 상환) · 대학원 석/박사
- 취업 15직군×회사 6형태(희귀도·스탯 게이트) · 업무평가 · 승진 10직급 · 연봉협상(역효과 리스크)
- 연간 랜덤 인생 이벤트 18종(복권·여행·첫사랑·스미싱…, 가족·자산 연동)
- 결혼(26~45세·취업·행복 55+) · 출산(최대 2명, 양육비) — 배우자·자녀가 방에 함께 삽니다
- 사고/질병/사망 리스크(단련된 몸은 피해 경감) · 12+ 열린 엔딩 · 로컬 명예의전당 · 결과 카드 이미지 공유
- **2세대 플레이**: 순자산 20% 유산 + 부모 스탯 재능 상속

**경제 루프**

- 방 꾸미기 14종(뽑으면 방에 렌더링) · 옷장 16종(입히면 스프라이트 복장이 실제로 바뀜)
  + 대형 자산(자동차 — 뽑기로 획득, 보유 티어보다 높은 차만 등장)
- 순자산(저축+자산)이 엔딩 점수 · 부자 판정 · 유산에 반영

**연출/편의**

- 시간대(새벽/낮/노을/밤)·계절(벚꽃/신록/낙엽/눈 낙하 애니메이션) 창밖, 행복 70+ 고양이
- 체중이 스프라이트 체형에 반영, 외형 60조합 랜덤(헤어 5종·톤·안경·볼포인트)
- 단색 LCD ↔ 컬러 도트 토글(🎨), 8비트 효과음(WebAudio, 🔊 토글)
- PWA 로컬 알림(탭 백그라운드 시 배고픔/피로/건강 경고), 게임 규칙 도움말 탭

## 🔐 데이터 / 개인정보 정책

- 회원가입 없음. 실명·전화번호·실제 회사/연봉 등 **개인정보를 일절 수집하지 않습니다.**

- 저장 레이어는 `lib/storage/` 에 추상화되어 있어, 이후 **Supabase Anonymous Auth + RLS**
  로 교체할 수 있습니다. (`lib/storage/local.ts` 의 TODO 참고)

## 🚀 실행 방법

```bash
npm install        # 의존성 설치
npm run dev        # 개발 서버 → http://localhost:3000
npm test           # 단위 테스트 (vitest, lib/game 순수 함수 162개)
npm run build      # 프로덕션 빌드
npm run start      # 프로덕션 실행
```

Node 18.18+ (권장 20+) 가 필요합니다.

## ☁️ Vercel 배포

1. 이 저장소를 GitHub 에 push (이미 `origin` 이 연결돼 있습니다).
2. [vercel.com](https://vercel.com) → New Project → 이 저장소 import.
3. 프레임워크 자동 감지(Next.js). 빌드 설정 그대로 두면 됩니다.
4. (선택) Environment Variables 에 `NEXT_PUBLIC_GAME_YEAR_MS` 추가 — `NEXT_PUBLIC_*` 은
   빌드타임에 박히므로 변경 후 **Redeploy 필요**.
5. Deploy. 별도 백엔드/DB 없이 정적+서버리스로 바로 배포됩니다.
