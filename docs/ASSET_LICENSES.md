# 에셋 라이선스 정책 (ASSET_LICENSES)

LifeGotchi 의 이미지/스프라이트/사운드 등 에셋 사용 정책과 라이선스 기록.

---

## 1. MVP 현황 — 외부 이미지 에셋 미사용

현재(MVP) 캐릭터·방·아이콘은 **전부 코드/CSS 로 생성**한다. 외부에서 내려받은
이미지·스프라이트·폰트 스프라이트·사운드 파일은 **사용하지 않는다.**

| 종류 | 출처 | 비고 |
|---|---|---|
| 사람형 픽셀 캐릭터 | 자체 제작(코드, `lib/game/sprite/*`) | div 그리드, 외부 이미지 0개 |
| 픽셀 방(PixelRoom) | 자체 제작(CSS) | 하드보더 박스 + 단색 면 |
| 8×8 픽셀 아이콘 | 자체 제작(`lib/pixel/icons.ts`) | 좌표 배열 |
| 마스코트(엔딩 카드) | 자체 제작(`components/character/Mascot.tsx`) | SVG/canvas |
| 폰트 | Pretendard / Galmuri (CDN) | 각 폰트 자체 라이선스 준수 |

→ 따라서 현재 저장소에 **제3자 이미지 에셋은 없다.**

---

## 2. 원작 Tamagotchi 리소스 — 사용 금지

다음은 **참고도 리소스로는 절대 사용하지 않는다.**

- 원작 Tamagotchi 스프라이트 / 캐릭터 외형
- 원작 Tamagotchi 사운드
- 원작 Tamagotchi 게임기(하드웨어) UI 이미지
- 타 게임/IP 스프라이트(포켓몬·쿠키런·메이플스토리 등 유사 포함)
- 구글 이미지 등에서 무단 다운로드한 캐릭터
- 저작권/라이선스가 불명확한 이미지

원작은 **감성·장르 참고만** 하고 실제 서비스 리소스로 쓰지 않는다.

---

## 3. 참고 가능한 에셋 사이트 (참고용 — 사용 전 개별 확인 필수)

아래는 **방향 참고용** 목록이며, 실제 사용 전 반드시 개별 라이선스를 확인하고
이 문서 §5 에 기록한다.

1. **Universal LPC Sprite Sheet Generator** — 사람 캐릭터 베이스/방향별 걷기/의상 구조
   참고. (생성기 내부 파츠별 라이선스 상이 — 개별 확인 필수)
2. **Pixeldex** — CC0 픽셀 스프라이트 참고(상업/리컬러 가능성 높음, 개별 확인).
3. **FreePixel** — 픽셀 캐릭터/타일셋/UI 참고(사용 전 확인).
4. **FreeGameSprites** — CC0 픽셀 게임 에셋 참고(사용 전 확인).
5. **OpenGameArt (CC0 필터)** — 다양한 CC0 캐릭터/타일셋(에셋별 라이선스 상이 — 확인).
6. **itch.io** (pixel art / virtual pet / tamagotchi 태그) — 가상펫/레트로 UI/픽셀 아이콘
   참고(유·무료/라이선스 혼재 — 무조건 확인).

## 4. 외부 에셋 사용 전 체크리스트

새 외부 에셋을 도입하려면 **모든 항목**을 통과해야 한다.

- [ ] 라이선스가 명시되어 있는가? (CC0 / CC-BY / MIT / 상업적 사용 허용 등)
- [ ] 상업적 사용이 허용되는가? (서비스/배포 가능)
- [ ] 수정·리컬러·2차 가공이 허용되는가?
- [ ] 저작자 표기(attribution) 요구가 있는가? 있으면 표기 위치를 정했는가?
- [ ] 원작 Tamagotchi / 타 IP 유래·유사 리소스가 아닌가?
- [ ] 출처 URL 과 라이선스 원문(또는 캡처)을 보관했는가?
- [ ] 이 문서 §5 표에 출처·라이선스·사용 범위를 기록했는가?

하나라도 불확실하면 **사용하지 않는다.**
